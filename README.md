# 🤖 Proxy Parser Bot

Telegram бот для парсинга и валидации прокси-серверов (SOCKS4, SOCKS5, HTTP, HTTPS).

## ✨ Возможности

- ✅ **Парсинг прокси** из множества источников
- ✅ **Валидация** IP адресов и портов
- ✅ **Проверка живых прокси** (проверка доступности)
- ✅ **Удаление дубликатов**
- ✅ **Поддержка несколько типов прокси** (SOCKS4, SOCKS5, HTTP, HTTPS)
- ✅ **Контроль доступа** по ID администраторов
- ✅ **Экспорт в файл** для больших списков

## 🚀 Быстрый старт

### Требования
- Python 3.9+
- Telegram Bot Token (получить от @BotFather)

### Установка

1. **Клонируем репозиторий**
```bash
git clone https://github.com/andrejzenuk9-ops/parser.git
cd parser
```

2. **Создаём виртуальное окружение**
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. **Устанавливаем зависимости**
```bash
pip install -r requirements.txt
```

4. **Настраиваем переменные окружения**

Создаём файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

Редактируем `.env` и добавляем:
```env
BOT_TOKEN=your_bot_token_here
ADMIN_IDS=123456789,987654321
```

**Или через GitHub Secrets:**
- Идёте в Settings → Secrets and variables → Actions
- Добавляете `BOT_TOKEN` и `ADMIN_IDS`

5. **Запускаем бота**
```bash
python bot.py
```

## 📖 Использование

### Команды бота
- `/start` - Начать работу, показать меню выбора типа прокси
- `/help` - Справка и информация

### Типы прокси
- **🔹 SOCKS4** - Протокол SOCKS версии 4
- **🔷 SOCKS5** - Протокол SOCKS версии 5 (рекомендуется)
- **🟢 HTTP** - Протокол HTTP
- **🟢 HTTPS** - Протокол HTTPS
- **⭐ Все типы** - Получить все доступные типы

## 🔧 Как это работает

### Процесс парсинга:

1. **Получение** - Бот получает данные с нескольких источников
2. **Валидация** - Проверяется формат IP и портов
3. **Проверка доступности** - Тестируется подключение к каждому прокси
4. **Вывод** - Отправляются только живые и валидные прокси

### Валидация прокси

```python
ProxyValidator.validate_socks4(proxy)  # Проверка SOCKS4
ProxyValidator.validate_socks5(proxy)  # Проверка SOCKS5
ProxyValidator.validate_http(proxy)    # Проверка HTTP/HTTPS
ProxyValidator.check_proxy_alive(proxy, proxy_type)  # Проверка доступности
```

## 🐛 Решение проблем

### Бот не запускается
```
Проверьте:
1. BOT_TOKEN правильно установлен
2. Интернет соединение работает
3. Версия Python >= 3.9
```

### Нет прокси в результатах
```
Возможные причины:
1. Источники недоступны
2. Все прокси неживые
3. Ошибка сети или timeout

Решение: попробуйте позже или выберите другой тип
```

### Ошибка "У вас нет доступа"
```
Добавьте ваш ID в ADMIN_IDS в переменных окружения
Узнать свой ID можно у @userinfobot
```
